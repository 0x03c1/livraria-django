# Programação para Web — Projeto Livraria com Django

---

## Objetivos de aprendizagem

Ao final deste material, o(a) aluno(a) será capaz de:

- Configurar um ambiente de desenvolvimento Python/Django profissional.
- Compreender a arquitetura **MTV** (Model–Template–View) do Django.
- Modelar entidades de domínio com a ORM do Django.
- Construir interfaces web utilizando o sistema de **templates** e **Bootstrap**.
- Implementar formulários com validação utilizando `ModelForm`.
- Desenvolver um **CRUD** completo (Create, Read, Update, Delete).
- Aplicar autenticação e controle de acesso.
- Realizar o **deploy** de uma aplicação Django.

## Stack utilizada

- **Python** 3.11+
- **Django** 5.x
- **SQLite** (desenvolvimento) / PostgreSQL (produção, opcional)
- **Bootstrap** 5
- **Git** + GitHub
- **VS Code** (editor recomendado)

---

## Como usar este material

1. Siga as aulas em ordem — cada uma depende da anterior.
2. Em cada aula, **digite o código manualmente** (não copie e cole). Isto faz parte do aprendizado.
3. Ao final de cada aula, faça um **commit** no Git com a mensagem sugerida.
4. As seções **Exercícios** ao final de cada aula são obrigatórias.
5. Em caso de dúvida, consulte a documentação oficial referenciada na [aula 99](./docs/aula99-referencias.md).

---

## Resultado final

Ao concluir todas as aulas, você terá uma aplicação web funcional com:

- Cadastro de livros, autores, categorias e editoras.
- Listagem com busca e paginação.
- Sistema de autenticação (login/logout).
- Painel administrativo customizado.
- Interface responsiva com Bootstrap 5.
- Aplicação em produção (deploy).
